# Intelligence Network · 全球智能网络

Intelligence Network maps the six node types in the emerging distributed AI-human research civilization — labs, individuals, agentic clusters, archives, simulators, governance councils — and the actual flows of insight between them. Bilingual (EN / 中文), dark/light, network-graph brand (Geist 600 + Inter Tight body + Geist Mono on `#08080d` with network-amber primary `#ffb04d`; Noto Sans SC for Chinese). Hero: animated SVG 6-node fully-connected graph with weighted edges. Three modules: N1 Six Node Types (cards × structural role), N2 Reader (5 multi-paragraph canned probes — where-knowledge-comes-from now, agentic-clusters reality vs hype, individual-rise structural reasons, governance-lag honest read, how-to-participate as multi-node — plus free-text heuristic), N3 Guardrails (no utopian decentralisation, no "academia is dead" framing, no agentic-cluster hype, no flat egalitarianism, no infinite-AI-generation slop).
